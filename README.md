# interview-task-3

## Pre-requisites
- Node 12 (LTS)
- yarn

## Getting started
- Check out
- Run `yarn install`

## Running locally
- Run `yarn start:dev` to run the dev version
    - Alternatively, in a Jetbrains IDE, run the `start:dev` run configuration
- Visit localhost:8000

## Deployment
- Not intended for deployment :)
